# Student Performance Analytics Dashboard

## Overview
This Power BI dashboard analyzes overall student pass percentage across departments and subjects. It helps identify low-performing and high-performing subjects using interactive visuals.

## Tools Used
- Power BI
- Excel (source data)

## Features
- Overall pass percentage KPI
- Subject-wise performance comparison
- Lowest to highest pass percentage ranking
- Academic year trend analysis
- Department and academic year slicers (AERO, AIDS, CSE, etc.)

## Dataset
The dataset contains subject-wise and department-wise pass percentage details across academic years.

## Files
- report for students (2).pbix – Power BI dashboard file  
- dashboard.pdf – Exported dashboard view

## How to View
1. Download the `.pbix` file.
2. Open it using **Power BI Desktop**.
3. Interact with filters and visuals.
